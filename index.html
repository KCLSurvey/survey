🔑 API-ключ для ИИ-анализа

Нужен для генерации выводов и рекомендаций. Сохраняется в браузере.

⚡ Способ 1 — Cloudflare Worker (рекомендуется для GitHub Pages):
1. Зайдите на cloudflare.com → бесплатный аккаунт
2. Workers & Pages → Create → вставьте код Worker
3. Задеплойте → скопируйте URL вида https://xxx.workers.dev/analyze
4. Вставьте URL ниже и нажмите Сохранить Worker

🔑 Способ 2 — API-ключ напрямую (резервный):
1. Зайдите на console.anthropic.com → API Keys → Create Key
2. Скопируйте ключ (начинается с sk-ant-) и вставьте ниже
